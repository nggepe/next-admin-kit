# Circle Bullet

This component just a circle. You just can set the color and the size of circle. The default color is using `var(--accent-a10)`.

## Default Example

```tsx
import CircleBullet from "@/components/bullet/CircleBullet";

const YourComponent = () => {
  return <CircleBullet />;
};

export default YourComponent;
```

## Using Size and Color

```tsx
import CircleBullet from "@/components/bullet/CircleBullet";

const YourComponent = () => {
  return <CircleBullet color='#000' size='12' />;
};

export default YourComponent;
```
